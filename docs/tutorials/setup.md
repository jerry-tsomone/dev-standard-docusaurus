# Setup

## Requirements

- Node.js 18+
- pnpm or npm

## Installation

Clone the boilerplate:

    git clone https://github.com/your-org/nextjs-boilerplate.git
    cd nextjs-boilerplate
    pnpm install

Start the dev server:

    pnpm dev

Visit [http://localhost:3000](http://localhost:3000).
